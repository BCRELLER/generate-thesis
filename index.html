USING PROCESSING AS A DESIGN TOOL
NUETROWAVES BY: JEROME HERR

WHAT IS PROCESSING?

USING CODE, PROCESSING IS A POWERFUL TOOL THAT BRINGS THE LOGICAL CONSTRUCTS OF COMPUTER PROGRAMMING TO A VISUAL MEDIUM.

Processing is great for graphic designers who are looking for new and powerful tools to express their ideas. It can be used for anything ranging from game design, infographics, creating music and generative art. These examples are only a few of the many uses of the Processing platform.

WHY IS IT IMPORTANT?

THE PROCESSING LANGUAGE IS BUILT OFF OF JAVA AND CAN AID IN LEARNING OF COMPUTER PROGRAMMING SKILLS.

Processing is not only a great tool for designers to create with; Processing also helps familiarize users with the basics of computer programming in the Java language. Not only will users be creating unique projects that only Processing can create, but gain valuable skills that will carry over to other programming languages.
